Simple Web Indexing System for Humans: C++ version

Introduction

SWISH++ is a Unix-based file indexing and searching engine (typically used to index and search files on web sites). It was based on SWISH-E although SWISH++ is a complete rewrite. SWISH++ was developed to circumvent my difficulties with using the SWISH-E package. SWISH++ has been ported to compile and run under Microsoft Windows by Robert J. Lebowitz and Christoph Conrad.

  • Features

    Lightning-fast indexing, handles large collections of files, tools included to index non-text files such as Microsoft Office documents, near instantaneous searches, and more...

  • Fully Documented

    Unix-style "man" pages are included describing the commands as well as the index file format. The source code is also generously commented. Additionally, sample Perl code for a web front-end is provided.

  • Download for Free

    SWISH++ is free software available under the terms of the GNU General Public License. It also happens to be free of cost.

    Unix requirements: A C++ compiler (such as the GNU C++ compiler), a version of STL (the C++ Standard Template Library), and GNU make.

    Microsoft Windows 98/NT/2000/XP requirements: The Cygwin software.

    Download generic source code:
    swishplusplus-6.1.5.tar.gz

    The change log is available.

  • FAQ and Freshmeat Entry

    There is an FAQ (Frequently Asked Questions, with answers) list. If you have a question, check it first. There is also a Freshmeat.net entry. If you want to notified only of new releases, you can be notified of them by subscribing there.
